# Break

A simple menubar app that reminds you to take a small break.

![break](./assets/break.png)

## About

The application is built with Tauri and Svelte. Tested on macos

## Development

To run the application locally, run:

```bash
pnpm tauri dev
```

```bash
pnpm tauri build
```
